## ADDRESSING THE REAL QUESTIONS

---

**"Will my account get banned?"**

Genuine automation risk comes from two things: (1) acting like a bot with no delays or limits, and (2) using mass-follow/unfollow schemes that violate Twitter's ToS.

This agent is built around neither of those. Every action has human-mimicking delays. Every daily limit is configurable and conservative by default. The browser automation uses Playwright Stealth to avoid detection fingerprints. The agent pauses itself when rate limits are hit.

Is there zero risk? No tool can promise that — Twitter's policies are always evolving. But the defaults in this tool are designed to look like a power user, not a bot.

**"I'm not technical — can I actually set this up?"**

Yes. That's why double-click setup files exist for both Mac and Windows. You double-click `2. Setup.command` (Mac) or `setup.bat` (Windows), wait 5 minutes, and the browser opens automatically. The only "technical" step is getting an Anthropic or OpenAI API key, which takes 2 minutes on their websites.

**"Won't it sound robotic?"**

You fill in a Voice Profile before the agent starts posting — your niche, your product, your personality, phrases you never use. The AI writes from that profile. You review every tweet in the Approval Queue before it posts (unless you turn off approval). You can regenerate any tweet you don't like with one click.

The more you tune the Voice Profile, the better the output gets.

**"What if I don't like it?"**
