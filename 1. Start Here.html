X-AUTOPILOT
SETUP GUIDE

Let's get you running.

Follow the steps below for your platform — takes about 5 minutes.

1

Unlock the files (one-time fix)

macOS blocked these files because they were downloaded from the internet. This is normal and not a security problem — it happens to every Mac app that isn't from the App Store. Run this one command to fix it:

$ xattr -cr "/path/to/twitter-agent 4"
1
Press ⌘ Space — type Terminal — press Enter
2
Paste the command above (⌘ V) and press Enter
3
The command runs silently — no output means it worked. Close Terminal.
⚡ You only need to do this once. After this, everything double-clicks normally.
2

Run setup (one-time only)

Right-click 2. Setup.command in the folder → click Open → click Open again in the popup.

RIGHT-CLICK MENU
Open
Open With…
Get Info
Move to Trash

A terminal window opens and installs everything. Takes 2–5 minutes. You'll see progress messages.

3

Launch the agent

Double-click 4. Run Agent.app any time you want to run it.

Your browser opens automatically to the dashboard:

http://localhost:5001
4

Add your API key

In the dashboard: go to the Settings tab → fill in your Anthropic API key and Twitter username → click Save.

✓ Get a free Anthropic key at console.anthropic.com — new accounts get free credits. OpenAI works too.
1
